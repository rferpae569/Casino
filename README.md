# Casino Rubén Fernández

Este proyecto fue generado con Angular CLI versión 16.0.4.

## Servidor de Desarrollo

Ejecuta ng serve para iniciar un servidor de desarrollo. Navega a http://localhost:4200/. La aplicación se recargará automáticamente si realizas cambios en alguno de los archivos de origen.

## Descripcion

Esta web esta hecha con la idea de que parezca un casino virtual. Al entrar, lo primero que veremos sera una web login como pagina principal y diversos apartados para acceder a las distintas paginas creadas, entre esas paginas se encuentran los juegos disponibles actualmente. Para acceder a los juegos se tendra que crear una cuenta o acceder con uno de los usuarios existentes de la base de datos.

Los juegos disponibles son:

- Maquina tragaperras.

- El juego de la ruleta.
